## 🚀 About AnimeNexus

AnimeNexus is a revolutionary cross-platform anime streaming application that brings together the best of modern technology and anime culture. Built with React Native for mobile platforms and Electron for desktop, AnimeNexus provides a seamless viewing experience across Android, Android TV, Windows, macOS, and Linux.

### ✨ Key Features

- 🎯 **Smart AI Recommendations** - Personalized anime suggestions based on your viewing history
- 📱 **True Cross-Platform** - Works seamlessly on phones, tablets, TVs, and desktops
- 🔄 **Multi-Device Sync** - Continue watching from where you left off on any device
- 📥 **Offline Viewing** - Download episodes for offline enjoyment
- 🏘️ **Community Features** - Rate, review, and discuss with fellow anime enthusiasts
- 🎨 **Customizable UI** - Multiple themes including dark mode and anime-themed designs
- 🔍 **Advanced Search** - Find anime by genre, year, studio, voice actors, and more
- 📊 **Watch Progress Tracking** - Keep track of what you've watched and plan to watch
- 🔔 **Episode Notifications** - Get notified when new episodes of your favorites are available
- 🌐 **Multiple Languages** - Support for multiple subtitle and audio languages

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or later)
- npm or yarn
- React Native CLI
- Android Studio (for Android development)
- Xcode (for iOS development, macOS only)

### Installation

```bash
# Clone the repository
git clone https://github.com/Darshil0305/AnimeNexus.git

# Navigate to project directory
cd AnimeNexus

# Install dependencies
npm install

# For mobile development
# iOS
npx react-native run-ios

# Android
npx react-native run-android

# For desktop development
# Start the Electron app
npm run electron
```
